# MindFuel 🌱

MindFuel is a lightweight web app that helps students track **mental energy and focus**, instead of just tasks.

## Why I built this
Most productivity tools ignore how a day actually feels.  
MindFuel captures focus, energy, mood, and reflection to help users understand patterns.

## Features
- Daily focus & energy tracking
- Mood selection
- Short reflection logging
- Weekly human-friendly insights
- LocalStorage based persistence

## Tech Stack
- HTML
- CSS
- JavaScript
- React (via CDN, component-based insights)

## What I learned
- DOM manipulation & event handling
- State management using LocalStorage
- Component-based thinking with React
- Writing human-centered software
